# ☕ Lista de Exercícios em Java (Lógica de Programação)
Este repositório contém uma série de exercícios essenciais desenvolvidos em Java, focados em Orientação a Objetos básica, tipos primitivos, entrada de dados com Scanner e formatação de saída que estudei durante o 2º semestre da graduação em ADS.

## O que está disponível neste repositório?
A lista aborda os mesmos conceitos lógicos, agora aplicados à sintaxe Java:

1. **Conversor de Moedas:** Uso de double para cálculos financeiros e Locale.US para leitura de pontos decimais.
2. **Troca de Variáveis:** Lógica de inversão de valores entre variáveis.
3. **Operações Matemáticas:** Uso da classe Math (Math.pow e Math.sqrt).
4. **Cálculo de Idade:** Manipulação de datas com Calendar ou LocalDate.
5. **Folha de Pagamento:** Operações aritméticas para descontos e acréscimos percentuais.
6. **O Maior de Três:** Implementação da fórmula matemática usando Math.abs.
7. **Gasto de Combustível:** Cálculo de consumo com saída formatada via printf.

## Tecnologias Utilizadas
* **Java JDK (versão 17 ou superior)**
* **Scanner Library:** Para entrada de dados via console.
* **Math Class:** Para operações de potência e raiz.

## Assinatura e Formatação
Os códigos prezam pela tipagem forte e organização em métodos, utilizando System.out.printf e formatadores como %.2f para garantir que os resultados financeiros e matemáticos sejam exibidos corretamente.
